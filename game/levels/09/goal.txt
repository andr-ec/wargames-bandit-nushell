Level Goal
----------
The password for the next level is stored in the file **data.txt**
in one of the few human-readable strings, preceded by several '='
characters.

Nushell commands you may need to solve this level
-------------------------------------------------
[open](https://www.nushell.sh/commands/docs/open.html),
[str trim](https://www.nushell.sh/commands/docs/str_trim.html),
[lines](https://www.nushell.sh/commands/docs/lines.html),
[where](https://www.nushell.sh/commands/docs/where.html)

You can also use external commands like `strings` to extract human-readable text from binary files.
