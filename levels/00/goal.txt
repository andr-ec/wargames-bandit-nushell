Level Goal
----------
The password for the next level is stored in a file called **readme**
located in the home directory.

Nushell commands you may need to solve this level
-------------------------------------------------
[ls](https://www.nushell.sh/commands/docs/ls.html),
[cd](https://www.nushell.sh/commands/docs/cd.html),
[open](https://www.nushell.sh/commands/docs/open.html)

**TIP:** Create a file for notes and passwords on your local machine!

Passwords also occasionally change. It is recommended to take notes on how to solve each challenge.
