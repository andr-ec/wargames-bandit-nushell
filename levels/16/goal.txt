## Level Goal

The password for the next level can be retrieved by submitting the
password of the current level to **port 30001 on localhost** using
SSL/TLS encryption.

Helpful note: Getting "DONE", "RENEGOTIATING" or "KEYUPDATE"?
Read the "CONNECTED COMMANDS" section in the openssl manpage.

## Nushell commands you may need to solve this level

[open](https://www.nushell.sh/commands/docs/open.html)

You can also use external commands like `openssl s_client` to connect to SSL ports.
